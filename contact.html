Why this page instead of a plain href with mailto?
After setting up a .onion url, I realized that typically Tor has JavaScript disabled using NoScript. Since I needed JavaScript for email obfuscation, I tried using the <noscript> tag, but NoScript apparently disables it by default as well. I dislike contact forms, so I figured setting up a separate page like this would take care of these problems.
