Hello!

My name is Hasan and here I write about my passions about such as Programming, GNU/Linux, free software, or torrenting. I like simple websites, so this website was stripped down to the bare-minimum.

This website also serves as sort of a "media hub". I didn't pay for a server just for this website! My favorite playlists, artists, programs, etc. will at some point be hosted here (if they're not already).

Projects:

  • aurinstall - An AUR helper written in Python. Currently being re-written in C, and will be renamed to caur.
  • hstatus - A simple status script written for dwm. Shows load average, ram usage, date, and time. No external libraries or dependancies.
  • simple-json - A simple yet slow JSON parser written in C. I use it when I need something to parse JSON without an external library (mostly for licensing reasons).
  • wordsearch-solver - A wordsearch solver written in Python. Can output to an image using PIL or using JSON.
  • dwm - My own customized version of the suckless dynamic window manager. Full credit goes to the suckless team, as I just applied patches.
